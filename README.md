# Análisis de Accidentes Automovilísticos

Este proyecto contiene un análisis exploratorio de datos (EDA) sobre un conjunto de datos de accidentes automovilísticos. El análisis se realiza mediante Python en un Jupyter Notebook, utilizando bibliotecas como **pandas**, **matplotlib** y **seaborn**.

## Objetivo

Identificar patrones, tendencias y posibles causas en accidentes de tránsito, con el fin de extraer conocimientos útiles para la toma de decisiones, prevención de accidentes o elaboración de reportes técnicos.

## Contenido

El análisis incluye:

- Carga y exploración inicial de datos.
- Limpieza de datos: tratamiento de valores nulos y formatos.
- Estadísticas descriptivas.
- Visualizaciones:
  - Distribución de accidentes por hora y día.
  - Relación entre tipo de accidente y clima.
  - Mapas de calor y gráficos de barras para correlaciones.
- Comentarios interpretativos sobre los resultados.

## Tecnologías utilizadas

- Python 3.x
- Jupyter Notebook
- pandas
- matplotlib
- seaborn
